Wikimedia Commons has media related to Cloud computing providers .
This is a set category . It should only contain pages that are Cloud computing providers or lists of Cloud computing providers , as well as subcategories containing those things (themselves set categories). Topics about Cloud computing providers in general should be placed in relevant topic categories .
For more information, see Cloud computing .

Cloud computing providers are notable entities who have verifiably significant production cloud computing service offerings.
